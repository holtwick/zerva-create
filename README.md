# create-zerva

Create a project using [Zerva](https://github.com/holtwick/zerva).

```bash
$ npm init zerva <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

If using Yarn:

```bash
$ yarn create zerva <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
